Universal Patch

    This is a universal patch that will fix problems with incorrect texture rendering, i.e. shaders, but it will not fix missing models and textures on maps in Half-Life 2 and episodes (to fix this you must install the steam_legacy version of Half-Life 2 as said before), errors related to Platform error, errors with Can't find background image and also when you have problems connecting to servers which tells you that the server is running a newer version of the game/older version of the game. Patch also fixes missing UI in CS:S build for PC, including Windows and MacOS. The patch contains hl2+platform from the free demo version of the game, but without maps and chapters, so it can be installed with overwriting existing files.

Server browser fix

    All master servers for browsing servers list included in the source engine 1.16 and 1.17 are down now. There's a new master server is running, but you need to manually add it to your game. To add it throw autoexec.cfg file to your game cfg files. For example path to the new cfg file should be srceng/cstrike/cfg/autoexec.cfg. If you already have this file, you probably want to manually edit it instead and add the new command there addmaster "ms.workbench.network:27010". This new address should now be working for a long time, because it doesn't rely on a specific IP address.
